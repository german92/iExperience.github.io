Exercises

E1: API FUN

Step 1: F

F is for friends, who do stuff together. Pick a partner or two who you've never worked with. Together, think of a cool use of one of the APIs we've learned, or find some others. Run your idea by a TA to make sure it's feasible.

Step 2: U

U is for u and me. To work together, we'll need a way to collaborate. Create a Github repo for your project. Add all group members as collaborators. Make sure each of you have a copy of the repo on your machine and can push changes to it.

Step 3: N

N is for 'n' get to work! Spec out, plan and pseudo-code your project. Divide up the tasks and write the code. Merge the changes, and test!
