E2: Petitions on Rails

We recently took the reddit Ruby app and turned it into a Rails application. Your task is to do the very same thing to our Petitions Ruby app. To do so, follow the steps:

  1. Follow the same steps from lesson 7B to take the Reddit Ruby app into a Rails app
  2. Follow the steps we did today to add forms for parameters to configure the feed (e.g. choosing different kinds of petitions such as open or closed)
For convenience, below is a list of steps reviewing what we did at a high level in 8B's lecture:

Code-Along: Reddit Rails Cont'd Steps

  1. Clone the reddit_rails repo
  2. Run the rails server and re-familiarize yourself with the app and how it works
  3. Go into the index view and add a form_tag and a select tag to select which feed you want ("top", "hot")
  4. Create a submit button and a new action (e.g. 'filter') to handle the submission from the form
  5. Add a new route for the POST action in your routes file
  6. Add a new method in your controller for the 'filter' action, to filter stories by the params
