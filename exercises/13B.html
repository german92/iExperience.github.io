Review

Topics Covered: Search; pagination; AJAX

Exercises

E1: Styled Up

Add some styling to the Prodix home page. In particular, make the alert and notice paragraphs look nice. Center the content, and change how the links.

E2: Search the Tweets

Add a search feature to the Working Twitter app to search through the tweets. Remember, you'll need to add form in the view, a controller action, and model method to get this to work.

Paginate

Add pagination to your tweets as well.

E3 (Bonus!): AJAX TWEETS

Make it so that you can Tweet via AJAX—that is, when you create a Tweet, the browser sends a request that goes to your controller, creates the Tweet, comes back with all the Tweets including the new one, after which the browser can render them—all without refreshing the page.
