E1: To 100

Write a program that prints the numbers from 1 to 100.

Hover to show answer 
100.times do |x|
  puts x
end

# Alternate solution
x = 1
while x <= 100
  puts x+1 # why x+1?
  x += 1   # shorthand for x = x + 1
end

E2: Fizzbuzz

Now write a program (also called a script) that prints the numbers from 1-100, but for each multiple of 3, print "fizz", each multiple of 5 print "buzz" and each multiple of 3 and 5 print "fizzbuzz": 

$ ruby fizzbuzz.rb
1
2
fizz
4
buzz
...
13
14
fizzbuzz
16
...

The modulo operator returns the remainder when the first input is divided by the second. That is, 10 modulo 4 would return 2. A quick way to check if a number is divisible by another is to see if their modulo is 0. In ruby, the % symbol is the modulo operator, that is 10 % 4 == 2.

Hover to show answer 
# fizzbuzz.rb
        
100.times do |x|
  x = x+1
  mult_3 = (x % 3 == 0)
  mult_5 = (x % 5 == 0)
  if mult_3 && mult_5
    puts "fizzbuzz"
  elsif mult_3
    puts "fizz"
  elsif mult_5
    puts "buzz"
  else
    puts x
  end
end

E3 (Bonus!): Reverse

Write a program that prints out the array ["apples", "bananas", "carrots", 4] in reverse. Don't just flip the array in the code. 

$ ruby reverse.rb
4
carrots
bananas
apples
Hover for answer 
# reverse.rb

list = ["apples", "bananas", "carrots", 4]
index = list.length - 1
while index >= 0
  puts list[index]
  index -= 1
end

# alternate (cheeky) solution
list = ["apples", "bananas", "carrots", 4]
list.reverse.each do |item|
  puts item
end

# If it's a short loop, you can do it on one line:
list.reverse.each { |item| puts item }
