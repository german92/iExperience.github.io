Review

Topics Covered: Rails views review; view helpers; partials

Exercises

E1: Poppin' Tags

We've learned so many tags in erb (note: different from irb). Let's do some review:

  1. When do you use <%= %> and when do you use <% %>?
  2. What does the yield in app/views/layouts/application.html do?

For each of the following tags, write down what it does, what parameters it takes and whether you need to put it in a <%= %> or <% %>.

  1. image_tag
  2. link_to
  3. render or render :partial =>
  4. form_for. What is special about form_for? What is that |f| thing it gives you?
  5. f.label
  6. f.text_field
  7. f.text_area
  8. f.select
  9. f.submit

E2: Retweet Again

Perfect practice makes perfect. Recreate the Twitter app again, from scratch, but now with:

  • Users that have profile pics
  • A header and a footer in the layout
  • Using at least one link_to and one image_tag
  • A form_for view helper for the User view
Try to look at the codealong from class as infrequently as possible.

E3: Linked up

Use link_to and image_tag to make an image that links somewhere. Make that link open in a new tab.

E4: Catch up

If you haven't had a chance to finish your shopping cart project or tic-tac-toe, make sure to finish them.
