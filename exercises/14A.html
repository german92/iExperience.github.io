Review

Topics Covered: Javascript; jQuery

Exercises

E1: Read This Note

JavaScript and jQuery are incredibly important parts of becoming a well-rounded web developer or designer. Any professional environment will expect you to at least be familiar with it, so even if you don't use Javascript in your final project, please take the time to learn a little bit now. In the worst case, you spend a bit of time learning a new language and making the coding part of your brain more versatile—that never hurt anyone!

E2: Practice

As you learn Javascript, you may be surprised at how much knowing Ruby helps you pick it up. You're saying the same things, just in a different dialect. Work through the intro Javascript track on Codecademy, and be able to do the last two modules, Build a Phone Book and Build a Cash Register. You can try them right away, or work up to them—just make sure you can pass those! Work through these quickly—if you find yourself spending more than an hour, move on.

E3: Learn jQuery

While Javascript is a programming language all on its own, jQuery adds a lot of functionality to it that makes is fantastic for the web. learn.jquery.com is where it starts: be able to complete the chapters "About jQuery", "Using jQuery Core", "Events", "Effects", and "AJAX".
